# JLPT N3 合格リビューア — Grammar・Vocab/Kanji・Reading

A complete reviewer for the bridge level. N3 is where the JLPT switches from "do you know the form?" to "can you tell two similar forms apart?" — and where three separate section minimums first apply. ~3,700 words, ~650 kanji.

---

## 0. Know the Scoring Game First

- **Total:** 180 points. **Pass line: 95/180 (~53%)** — higher than N2's 90. N3 is not a soft target.
- From N3 up there are **THREE scoring sections** of 60 points each: Language Knowledge, Reading, Listening — each with a **19/60 minimum**. You can no longer pool vocab/grammar with reading.
- Translation: Reading becomes a section that can fail you on its own. Most N3 fails are reading-section fails.
- The JLPT uses scaled scoring. **Never leave blanks** — no guessing penalty.
- Test format: Vocabulary **30 min** → Grammar・Reading **70 min** → Listening **40 min**.

---

## 1. 文法 (Grammar) — Where Nuance Starts

N3 grammar questions come in three flavors:
1. **文法形式の判断** — pick the right grammar for the blank (~13 questions)
2. **文の組み立て (★問題)** — reorder 4 fragments; answer the one in the ★ slot (~5 questions)
3. **文章の文法** — cloze passage (~5 questions)

### 1.1 The Confusable Pairs (JLPT's favorite N3 traps)

**The わけ family (tested every single session)**
- わけだ = "so that means / no wonder" (logical conclusion): 3年も日本にいたのか。日本語が上手な**わけだ**。
- わけではない = "it's not that..." (partial denial): 嫌いな**わけではない**が、あまり食べない。
- わけがない = "there's no way" (strong denial): 彼が知らない**わけがない**。
- わけにはいかない = "can't (for social/moral reasons)": 大事な会議だから、休む**わけにはいかない**。

**〜はずだ vs 〜はずがない vs 〜はずだった**
- はずだ = should be (logical expectation) / はずがない = can't possibly be / はずだった = was supposed to (but didn't): 来る**はずだった**のに、来なかった。

**〜ように vs 〜ために (purpose)**
- ように + non-volitional/potential/negative: 忘れない**ように**メモする。聞こえる**ように**大きい声で話す。
- ために + volitional verb, same subject: 留学する**ために**、お金をためている。

**〜ばかり family**
- 〜たばかり = just did: 食べた**ばかり**だ。
- 〜てばかりいる = does nothing but: 遊んで**ばかりいる**。
- 〜ばかりでなく〜も = not only... but also: 子ども**ばかりでなく**大人**も**楽しめる。

**〜ところ trio (form decides meaning)**
- するところ = about to / しているところ = in the middle of / したところ = just finished: 今から出かける**ところ**だ。

**〜うちに vs 〜間(に)**
- うちに = before the state changes (urgency): 熱い**うちに**食べて。暗くならない**うちに**帰ろう。
- 間に = within a defined span: 昼休みの**間に**銀行へ行く。

**〜たびに vs 〜と (repetition)**
- たびに = every time (with noun+の or dict form): この写真を見る**たびに**、国を思い出す。

**〜によって (4 uses — all tested)**
1. Means: インターネット**によって**調べる。
2. Cause: 地震**によって**ビルが倒れた。
3. Agent of passive: この寺は10世紀に**よって**建てられた→(人物)によって。
4. Depending on: 人**によって**考え方が違う。／ 場合**によっては**中止する。

**〜せいで vs 〜おかげで vs 〜ため(に)**
- せいで = bad result + blame / おかげで = good result (or sarcasm) / ために = neutral cause: 事故の**ため**、電車が遅れています。

**〜って (casual quotation — N3 loves it)**
- = という/と言っていた/とは: 田中さん、来ない**って**。(=来ないと言っていた)

**〜がる / 〜がっている**
- 3rd person's feelings need がる: 妹は犬を怖**がっている**。(✕ 妹は怖い)

**〜みたい vs 〜らしい vs 〜っぽい**
- みたい = looks like/inference (casual ようだ) / らしい = hearsay or "typical of" / っぽい = -ish tendency: 忘れ**っぽい**、子ども**っぽい**。

**〜ことになっている vs 〜ことにしている**
- ことになっている = rule/custom (decided by others): 制服を着る**ことになっている**。
- ことにしている = personal habit I decided: 毎朝走る**ことにしている**。

**〜てほしい vs 〜たい vs 〜てもらいたい**
- たい = I want to do / てほしい・てもらいたい = I want SOMEONE ELSE to do: 手伝っ**てほしい**。

**〜ば〜ほど**
- The more... the more: 練習すれ**ば**する**ほど**、上手になる。(same verb twice — a scramble-question favorite)

**〜さえ / 〜さえ〜ば**
- さえ = even: 子ども**でさえ**知っている。
- さえ〜ば = if only: 時間**さえ**あれ**ば**、行けるのに。

**〜たとえ〜ても / いくら〜ても**
- Even if (hypothetical): **たとえ**雨が降っ**ても**、行く。

**Keigo (honorifics) — N3 tests full sets**
- Respect: いらっしゃる、おっしゃる、召し上がる、ご覧になる、なさる、くださる、お〜になる、〜れる/られる (honorific use!)
- Humble: 参る、申す、いただく、拝見する、伺う (ask/visit/hear)、いたす、お〜する、〜させていただく
- ★Trap: passive-form-as-honorific (社長が話**される**) vs actual passive — context decides.

### 1.2 High-Frequency N3 Grammar Checklist

Anything you can't produce an example sentence for, drill it today:

**Cause/Reason:** 〜おかげで、〜せいで、〜ため(に)、〜によって、〜から〜にかけて、〜ことから、〜もの (casual excuse)、〜んだもの
**Contrast/Concession:** 〜のに、〜ても、〜ながら(も)、〜くせに (blame!)、〜わりに、〜としても、〜ところが、〜のに対して
**Time/Sequence:** 〜たとたん、〜(よ)うとしたとき、〜うちに、〜間に、〜たびに、〜ところに/ところへ、〜最中に、〜てからでないと、〜まで(に)
**Extent/Degree:** 〜ほど、〜くらい/ぐらい、〜ば〜ほど、〜すぎる、〜てたまらない、〜てしょうがない、数量+も、〜しか〜ない
**Decision/Change:** 〜ことにする/ことになる、〜ようにする/ようになる、〜つつある (fringe)、〜一方だ (fringe N2)
**Modality/Judgment:** 〜はずだ、〜に違いない、〜かもしれない、〜だろう/でしょう、〜みたい、〜らしい、〜ようだ、〜そうだ (both!)、〜べきだ、〜ことはない (no need)、〜ないことはない
**Obligation/Permission:** 〜なければならない、〜なくてはいけない、〜ないと (casual)、〜ずに、〜なくて済む、〜てはならない
**Giving/Receiving (advanced):** 〜てくださる、〜ていただく、〜てやる、〜てもらえませんか、〜ていただけませんか
**Formal/Written:** 〜における、〜に関する、〜に対する、〜として、〜とともに、〜を通じて、〜向け (fringe)、〜による

### 1.3 ★並べ替え (Sentence Scramble) Technique

1. Read the frame first — words before the first blank and after the last blank fix the grammar.
2. Find glued pairs: ば+ほど、た+とたん、という+noun、かどうか+verb. Grammar points weld two fragments.
3. Assemble the full chain, THEN check which fragment lands on ★.
4. Stuck >75 seconds → best guess, circle it, move on. One scramble = one point; a 長文 question is worth the same.

## 2. 語彙・漢字 (Vocab & Kanji) — ~3,700 Words, ~650 Kanji

### 2.2 表記 (Orthography)

### 2.3 文脈規定 (Fill the Context) — high-yield groups

- **Adverbs:** ぐっすり (sound asleep)、うっかり (carelessly)、しっかり (firmly/properly)、はっきり (clearly)、ゆっくり (slowly)、こっそり (secretly)、ぴったり (exactly/perfect fit)、そっくり (identical)、さっぱり〜ない (not at all)、まったく (utterly)、ずいぶん (quite)、かなり (fairly)、めったに〜ない (rarely)、案外 (unexpectedly)、意外に (surprisingly)、突然 (suddenly)、急に (abruptly)、次第に (gradually)、徐々に (little by little)、同時に (at the same time)、相変わらず (as always)、とりあえず (for now)、いよいよ (at last)、ますます (more and more)、わざと (on purpose) vs わざわざ (going out of one's way)

### 2.4 言い換え類義 (Paraphrase) — high-yield sets

- 徐々に ≒ だんだん (gradually) / 直ちに ≒ すぐに (immediately) / あらゆる ≒ すべての (every) / 主に ≒ ほとんど・大部分 (mainly)
- くたびれる ≒ 疲れる (to get tired) / びしょぬれ ≒ とてもぬれている (soaked) / まれに ≒ めったにない (rarely)
- 検討する ≒ よく考える (to consider carefully) / 回復する ≒ よくなる (to recover) / 縮小する ≒ 小さくする (to shrink) / 拡大する ≒ 大きくする (to expand)

- おそらく ≒ たぶん (probably) / 常に ≒ いつも (always) / 各自 ≒ それぞれ (each person) / 至急 ≒ 大急ぎで (urgently)

### 2.5 用法 (Usage) — 5 questions, the hardest vocab type

Four sentences, one correct use. Ask: **what does this word collocate with — people, objects, or situations?**
- 「募集」= organizations recruit people/entries ✓, not looking for lost keys ✕ (that's 探す)
- 「渋滞」= roads/traffic ✓, not crowded trains ✕ (that's 混雑)
- **Frequently tested:** 迎える(むかえる — to welcome)、届く(とどく — to reach)、伝わる(つたわる — to be conveyed)、似合う(にあう — to suit)、間に合う(まにあう — to be in time)、片付ける(かたづける — to tidy up)、たまる (to pile up)、蒸し暑い(むしあつい — muggy)、詳しい(くわしい — detailed/well-versed)、豊か(ゆたか — rich/abundant)、盛ん(さかん — thriving)、真剣(しんけん — serious)、順調(じゅんちょう — going smoothly)、意外(いがい — unexpected)、活躍(かつやく — playing an active role)、普及(ふきゅう — spreading/diffusion)、募集(ぼしゅう — recruitment)、節約(せつやく — saving)、検討(けんとう — consideration)、対象(たいしょう — target/subject)

## 3. 読解 (Reading) — The Section That Fails People

From N3 up, Reading is its own 60-point section with a 19-point floor — and it shares 70 minutes with grammar. Opinions start being *implied*, not just stated.

### 3.1 Question Types & Order of Attack

| 問題 | Type | Count | Strategy |
|---|---|---|---|
| 4 | 短文 (~200字) | 4 | 1 question each. Answer usually in the LAST 1–2 sentences. |
| 5 | 中文 (~350字) | 6 (2×3) | Read question stems FIRST, then the passage once. |
| 6 | 長文 (~550字) | 4 | Essay/explanation. Paragraph 1 = topic, last paragraph = author's point. |
| 7 | 情報検索 (ads, notices) | 2 | Question first, then scan. Fine print (※、ただし、締め切り) decides. |

**Do 問題7 (情報検索) FIRST if time-pressured** — pure scanning, 2 points, 4 minutes max.

### 3.2 The Golden Rules

1. **The author's words > your common sense.** Wrong answers are true-in-real-life but unstated.
2. **Reversal markers point at answers:** しかし、ところが、でも、実は — the sentence AFTER these carries the claim.
3. **〜のではないだろうか / 〜と思う / 〜べきだ = the author's opinion.** For 長文, the final paragraph almost always restates it.
4. **それ/これ/この指示語 questions:** referent is immediately BEFORE. Substitute back to verify.
5. **下線部 questions:** read 2 sentences before + 1 after the underline; the answer paraphrases that local context.
6. **Extreme answer choices are usually wrong:** 必ず、絶対、全部、まったく〜ない. Correct answers hedge like the passage.
7. **どうして questions:** hunt the から/ので/ため — reasons are grammatically marked.

### 3.3 Time Budget (70-min Grammar・Reading block)

- 文法 (問題1–3): **20 min**
- 短文・中文 (問題4–5): **26 min**
- 長文 (問題6): **10 min**
- 情報検索 (問題7): **4 min**
- Buffer / mark-and-return: **10 min**

Stuck >90 seconds: mark a guess, circle it, move on. Return only with buffer time.

## 4. The Final 48 Hours

**Do:**
- Re-drill Section 1.1 (the わけ family, ように/ために, ことにする/ことになる 2×2, keigo sets) — the highest-frequency traps.
- One pass of the kanji trap list (2.1) out loud, readings only.
- One timed 読解-only set to calibrate pacing — not a full mock this late.
- Sleep normally both nights; sleep debt hits reading hardest.

**Don't:**
- Learn brand-new grammar the day before.
- Batch-transfer answers at the end — fill as you go.

**Test-day mechanics:**
- Analog watch (venues often lack clocks); phones off.
- Guess every remaining blank when 2 minutes are called.
