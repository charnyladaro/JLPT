# JLPT 道場 — Study Dojo

By [Charnyl Adaro](https://github.com/charnyladaro)

A free, offline JLPT study app for all levels (N5–N1). Drill grammar with native
example sentences, study vocabulary with flashcards, and practice thousands of
exam-style questions — no account, no ads, no internet required.

Available as:

- **Android app** (`Android/`) — built with Expo, runs fully offline
- **Web app** (`webapp/`) — open `webapp/index.html` in any browser, no install needed

## Install (Android)

1. Download the APK from the latest release:
   **[JLPTdojo v1.0.0](https://github.com/charnyladaro/JLPT/releases/tag/v1.0.0)** →
   [`JLPTdojo.v1.0.0.apk`](https://github.com/charnyladaro/JLPT/releases/download/v1.0.0/JLPTdojo.v1.0.0.apk) (69 MB)
2. Open the downloaded file on your phone. If prompted, allow your browser or
   file manager to *install unknown apps* (Settings → Apps → Special access).
3. Tap **Install** and open **JLPT 道場**.

The app notifies you in the top bar when a newer release is available here on GitHub.
